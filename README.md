# programmers-solve

### 문제 업로드 방식
1. 한사람씩 돌아가며 말한 문제를 푼다.(이름순) 문제를 올리는 사람은 오전 10시까지 문제를 올린다. 문제는 디스코드에 올린다.
2. 월/수/금 - 매일 2문제 풀고 pull request 날린다. 저장 위치는 각자 개인 폴더.
3. 화/목 - 코드리뷰 comment 달기

```
📁 강수정
  └── [프로그래머스 문제 이름].js
📁 박다은
📁 서현주
📁 안현수
📁 오혜윤
📁 이재호
📁 허재원
```

### 코드리뷰 방식
- 각자 이니셜 브랜치를 판다. (ex. 현주 - hj)
  ```
  명령어는 제가 따로 올리겠습니다!
  ```

### 커밋 메세지
```
Solve : 문제 해결
Docs : 문서 수정
Style : 공백, 세미콜론 등 스타일 수정
```
