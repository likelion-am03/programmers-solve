# programmers-solve

### 코드리뷰 방식
1. 한사람씩 돌아가며 월/수/금 오전 10시까지 2문제를 디스코드에 올린다.
2. 월/수/금 - 매일 2문제 풀고 pull request 날린다. 저장 위치는 각자 개인 폴더.
3. 화/목 - 코드리뷰 comment 달기

### 폴더 구조
```
📁 강수정
  └── [프로그래머스 문제 이름].js
📁 박다은
📁 서현주
📁 안현수
📁 오혜윤
📁 이재호
📁 허재원
```

### 커밋 메세지
- 커밋 메세지는 아래의 태그를 참고하여 다음과 같이 작성한다.
```
Solve: 두 수의 차 문제 해결
```
- 태그
```
Solve : 문제 해결
Docs : 문서 수정
Style : 공백, 세미콜론 등 스타일 수정
```
